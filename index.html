시작하기
이 도구는 경계값 분석(BVA) 기법을 이용한 테스트 문자열 및 비밀번호 테스트 케이스를 자동으로 생성합니다. 별도 설치 없이 브라우저에서 바로 사용할 수 있습니다.
경계값 분석(BVA)란?
입력값의 경계 근처에서 버그가 많이 발생한다는 원리를 이용해, N−1 / N / N+1 세 가지 케이스를 테스트하는 기법입니다.
🔠 문자열 생성기
기준 글자 수를 입력하면 경계값에 해당하는 문자열을 자동 생성합니다.
항목설명
문자 종류한글 · 영문 대문자 · 영문 소문자 · 숫자 등 복수 선택 가능
기준 글자 수최대 허용 글자 수 (1 이상 정수)
출력 모드BVA 3세트 또는 단일 출력 선택
BVA 3세트 모드 생성 케이스:
케이스글자 수
미만N − 1자
정확N자 (경계값)
초과N + 1자
이모티콘·한글 같은 멀티바이트 문자는 글자 수 ≠ 바이트 수일 수 있습니다. 카드에서 두 값을 모두 확인하세요.
🔐 비밀번호 생성기
비밀번호 유효성 검사 테스트용 정상/비정상 케이스 세트를 생성합니다.
케이스설명
정상모든 조건 충족
비정상길이 부족 (N−1자)
비정상대문자로만 구성
비정상소문자로만 구성
비정상특수기호 미포함
특수문자(!) 옵션 선택 시 비밀번호 맨 끝에 !가 고정 포함됩니다.
공백 포함 선택 시 중간 임의 위치에 공백 1개가 삽입됩니다.
