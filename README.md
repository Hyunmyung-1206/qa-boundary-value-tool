# QA 경계값 테스트 입력값 생성기

QA 테스트 과정에서 경계값 입력 데이터를 더 빠르게 만들기 위해 제작한 보조 도구입니다.

## 제작 배경

프로젝트 테스트 중 300자, 1000자, 2000자 등 다양한 길이의 입력값을 반복해서 준비해야 했습니다.
처음에는 메모장에 직접 문자를 조합하고 글자 수를 확인했지만, 입력값 길이를 매번 측정해야 해 시간이 오래 걸렸습니다.
이를 줄이기 위해 글자 수와 문자 조건을 선택하면 테스트 입력값을 바로 생성할 수 있는 도구를 만들었습니다.

## 주요 기능

- 원하는 글자 수에 맞는 입력값 생성
- 한글, 영문 대문자, 영문 소문자, 숫자, 특수문자 등 조합 선택
- 경계값 및 예외 입력값 검증에 활용 가능한 테스트 데이터 생성
- 수동 테스트 시 입력값 준비 시간 단축

## QA 활용 의미

이 도구는 수동 검증 과정에서 반복되는 테스트 데이터 준비를 줄이기 위해 만든 사이드 프로젝트입니다.
Selenium과 직접 연동하는 방식도 고려했지만, 연동 작업이 수동 테스트보다 더 큰 공수를 만들 수 있다고 판단해 독립형 HTML 도구로 사용했습니다.

이를 통해 테스트 목적에 맞는 도구를 직접 만들고, 자동화 적용 여부를 공수와 효율 기준으로 판단하는 경험을 했습니다.
